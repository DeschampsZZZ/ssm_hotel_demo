# demo    酒店管理系统

## 概述
 本系统的后台运用了SSM框架，前端框架adminlte，数据库使用mysql。使用Maven管理jar包，使用Spring Security做权限管理，使用PageHelper分页插件做分页

## 包含功能

- 入住订单的添加、修改、删除、模糊查询，添加订单时修改房间状态。 
- 房间的增加、修改、删除、模糊查询。
- 分配不同用户登录角色不同权限。
